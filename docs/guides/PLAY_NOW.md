## Commands

| Command | Description |
|---------|-------------|
| `talk [name]` | Start talking to an NPC (e.g., `talk mara`) |
| `look` | Get atmospheric description of location |
| `npcs` | List all available NPCs |
| `info [name]` | View detailed info about an NPC |
| `where` | Show your current location |
| `stats` | View session statistics |
| `help` | Show all commands |
| `exit` | Quit the game |

## Troubleshooting

### "Connection refused" or slow responses
```bash
# Make sure Ollama is running
ollama serve

# In another terminal
ollama pull llama3.1:8b
```

### Game not starting
```bash
# Install dependencies
npm install

# Try again
node play-with-gm.js
```

### NPCs not responding
- Wait 10-20 seconds for LLM response
- Check that Ollama is running
- Try `ollama list` to see available models
